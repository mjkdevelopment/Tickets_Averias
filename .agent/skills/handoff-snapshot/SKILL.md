    # Handoff Snapshot

## Include
- Goal + current status
- Branch + commit(s)
- Files changed (high level)
- Commands run + results
- How to verify
- Known issues / risks
- Next steps (prioritized)

## Format
Use short sections and bullet points. Keep it paste-ready for Slack/Notion/next agent.
